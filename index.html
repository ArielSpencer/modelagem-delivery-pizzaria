Projeto Pizzaria Delivery

Este projeto apresenta a modelagem de dados para um sistema de pizzaria Delivery.

O objetivo é organizar as informações e facilitar a gestão de cadastros, produtos (pizzas e bebidas), clientes, pedidos, itens de pedido e motoqueiros em um sistema de entrega online.
